# Zero Lua

## Development

### Generate compile_commands.json (Linux and macOS)

This step is required for [clangd](https://clangd.llvm.org/) to work properly. Install [Bear](https://github.com/rizsotto/Bear) then run the following command:

```sh
bear -- cargo build
```
